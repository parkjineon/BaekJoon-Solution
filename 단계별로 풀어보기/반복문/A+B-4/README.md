<h1>EOF</h1>
<p> 

  End of File. 더 이상 읽을 수 있는 데이터가 없으며, 파일의 끝을 나타낸다. 
  
</p>

<h2>C++</h2>

<h3>cin.eof()</h3>
<ul>
  <li>cin으로 입력을 받으려고 하는데, EOF라면 입력은 멈추며, True를 반환한다.</li>
  <li>마지막 문자를 읽은 다음, <b>한 번 더 읽기을 시도했을 </b> 읽을 데이터가 없을 경우 EOF라고 판단한다.</li>
 </ul>

<h2>Python</h2>

<h3>try...except</h3>
<ul>
  <li>파일 끝에서 읽으려고 할 때, EORError 발생</li>
 </ul>
